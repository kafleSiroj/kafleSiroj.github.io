Resources I Used:

  • CS50
    • CS50 is a great free course provided by Harvard University. Prof. David J. Malan, the professor of CS50, teaches the foundations of coding, including C, data structures and algorithms (DSA), web development, AI, and SQL.

  • CS50 AI
    • CS50 AI is a branch of CS50 taught by Prof. Brian Yu. This course is designed to provide foundational knowledge on AI topics.

  • Understanding Deep Learning (Book)
    • Understanding Deep Learning (UDL) is an awesome book by Simon J. D. Prince. It's one of the best resources if you want to gain a deep understanding of deep learning. The book explains the mathematics behind the models and covers both supervised and unsupervised learning topics. It also provides Jupyter notebooks on its GitHub page, which contain exercises related to each chapter. At the end of every chapter, there is a problem set to reinforce what you've learned.
